# 视频/图文解析下载工具 V1 — 功能说明

## 项目位置
`C:\Users\YBY\Desktop\video-downloader\`

## 启动方式
```bash
# 后端
cd backend && python main.py
# 前端（另一个终端）
cd frontend && npm run dev
```
访问 `http://localhost:3000`

---

## 技术栈
- 前端: Vue3 + Vite + Axios（单页应用）
- 后端: Python FastAPI + requests
- 解析API: `https://api.bugpk.com/api/bilibili` 和 `https://api.bugpk.com/api/douyin`

---

## 已实现功能

### 1. 链接解析
- 支持B站链接（完整URL `bilibili.com/video/BVxxx` 和短链 `b23.tv/xxx`）
- 支持抖音链接（`v.douyin.com/xxx`）
- 自动去除链接中的 query 参数（`?share_source=...`）
- 自动从粘贴文本中提取 URL（即使带了标题文字一起粘贴）
- API 请求失败时有重试逻辑（2次），最终失败则返回模拟数据保流程可跑通

### 2. 资源下载
- 后端异步下载到本地指定目录
- 下载进度条实时显示百分比
- 支持单文件下载和批量下载全部
- 防盗链处理（添加 User-Agent、Referer 请求头）
- B站 CDN 链接需要防盗链绕过，抖音链接直接可用

### 3. 前端页面
```
┌──────────────────────────────────────────┐
│  视频/图文解析下载工具                      │
├──────────────────────────────────────────┤
│  下载目录: /path        [更改]             │
├──────────────────────────────────────────┤
│  [粘贴抖音或B站分享链接...]    [解析]      │
├──────────────────────────────────────────┤
│  封面图（自动展示，可点击放大）            │
│  标题文字                                 │
│  ☑ 封面缩略图 + 视频标题 + 文件大小 [下载] │
│  ☑ 封面缩略图 + 视频标题 + 文件大小 [下载] │
│  ┌──── 进度条 ──── 60% ──┐               │
│  [下载选中]  [下载全部]                   │
└──────────────────────────────────────────┘
```

- 封面图自动展示（通过后端 proxy 绕过防盗链）
- 每个文件前有复选框，支持多选下载
- 下载进度条（实时百分比）
- 点击封面/图片可全屏预览
- 视频标题可点击预览播放

### 4. 后端接口
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/parse` | 解析链接，返回标题/类型/封面/文件列表 |
| POST | `/download` | 开始下载，返回 task_id |
| GET | `/download/progress/{task_id}` | 轮询下载进度 |
| GET | `/download_dir` | 获取当前下载目录 |
| POST | `/set_download_dir` | 设置下载目录 |
| GET | `/proxy/image?url=...` | 代理加载图片（带 Referer） |

## V1 限制（V2 可以考虑改进的）
1. 没有 UI 美化（原生风格）
2. 没有登录/用户系统
3. 没有数据库
4. B站视频直链有防盗链，下载可能失败
5. 解析速度受第三方 API 速度影响（3-5秒）
6. 不支持图片下载（只做视频）
7. 不支持视频预览在线播放（B站直链过期快）
8. 没有下载任务队列管理
9. 没有 Electron 打包
10. 没有多平台扩展
